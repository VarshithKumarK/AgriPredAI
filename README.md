## ğŸ“¥ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/YourUsername/AgriPredAI.git
cd AgriPredAI
```

### 2. Setup ML Service (Python)

Navigate to the ML folder and install dependencies.

```bash
cd ML
# Create virtual environment (optional but recommended)
python -m venv venv
# Activate: venv\Scripts\activate (Windows) or source venv/bin/activate (Mac/Linux)

pip install -r requirements.txt
# Ensure you have tensorflow, flask, pillow, requests
```

**Setup Ollama:**

1.  Install Ollama.
2.  Pull the Llama 3.2 model:
    ```bash
    ollama pull llama3.2
    ```
3.  Start the Ollama server:
    ```bash
    ollama serve
    ```

**Start Flask API:**

```bash
python app/main.py
# Runs on http://localhost:8000
```

### 3. Setup Backend (Node.js)

Navigate to the backend folder.

```bash
cd ../backend
npm install
```

**Environment Variables:**
Create a `.env` file in `/backend`:

```env
MONGO_URI=mongodb://localhost:27017/crop_prediction
JWT_SECRET=your_super_secret_key
PORT=5000
```

**Start Backend Server:**

```bash
npm start
# Runs on http://localhost:5000
```

### 4. Setup Frontend (React)

Navigate to the frontend folder.

```bash
cd ../frontend
npm install
```

**Start Client:**

```bash
npm run dev
# Runs on http://localhost:5173
```
